# LCC_3: App Configuration System

## Description
Create a centralized configuration system for app-wide settings including default passwords, barista names, and elder-friendly design constants.

## Acceptance Criteria
- [ ] Configuration file created at `/src/config/app.ts`
- [ ] Default admin password is configurable (default: "admin123")
- [ ] Default barista names array is defined (minimum 3 names)
- [ ] Elder-friendly design constants are defined:
  - Minimum touch target size (44pt)
  - Large font sizes (minimum 18pt for body text)
  - High contrast color palette
  - Animation duration settings (slower for accessibility)
- [ ] App version and build information
- [ ] Configuration is typed with TypeScript interfaces
- [ ] Configuration values can be easily modified without code changes to other files
