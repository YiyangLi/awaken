# LCC_1: Project Setup and TypeScript Configuration

## Description
Set up the foundational project structure with TypeScript configuration, ESLint/Prettier rules, and basic folder architecture for the Awaken coffee ordering app.

## Acceptance Criteria
- [ ] TypeScript is configured with strict mode enabled
- [ ] ESLint and Prettier are configured with React Native and TypeScript rules
- [ ] Project folder structure is created:
  - `/src/components/` (reusable UI components)
  - `/src/screens/` (screen components)
  - `/src/types/` (TypeScript interfaces and types)
  - `/src/utils/` (utility functions)
  - `/src/config/` (app configuration)
  - `/src/storage/` (data persistence logic)
  - `/src/hooks/` (custom React hooks)
- [ ] Package.json scripts are configured for development and building
- [ ] All files compile without TypeScript errors
- [ ] ESLint runs without errors on initial setup
