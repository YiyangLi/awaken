# LCC_11: Modal System for Confirmations

## Description
Implement a modal system for confirmation dialogs, alerts, and forms with elder-friendly design patterns and clear action buttons.

## Acceptance Criteria
- [ ] Modal provider component in `/src/components/ModalProvider.tsx`
- [ ] Modal hook for easy usage: `useModal()`
- [ ] Confirmation modal with large Yes/No buttons
- [ ] Alert modal for important messages
- [ ] Loading modal with clear progress indicators
- [ ] Form modal for data input
- [ ] Modal backdrop with high contrast
- [ ] Large close buttons (X) in consistent locations
- [ ] Keyboard navigation support
- [ ] Auto-focus management for accessibility
- [ ] Customizable button text and colors
- [ ] Modal stacking management
