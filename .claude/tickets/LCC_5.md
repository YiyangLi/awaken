# LCC_5: Elder-Friendly Theme System

## Description
Create a comprehensive theme system optimized for elderly users with large fonts, high contrast colors, and accessibility-focused design tokens.

## Acceptance Criteria
- [ ] Theme configuration file created at `/src/config/theme.ts`
- [ ] Color palette includes:
  - High contrast combinations (minimum 4.5:1 ratio)
  - Primary, secondary, and accent colors
  - Success, warning, and error states
  - Background and surface colors
- [ ] Typography scale with large, readable fonts:
  - Heading sizes: 28pt, 24pt, 20pt
  - Body text: 18pt minimum
  - Caption text: 16pt minimum
- [ ] Spacing system with generous touch targets (minimum 44pt)
- [ ] Shadow and elevation definitions for depth perception
- [ ] Theme is consumable via React Context
- [ ] TypeScript interfaces for all theme properties
- [ ] Dark mode support for different lighting conditions
